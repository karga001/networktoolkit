# Network Toolkit

Basit ve etkili ağ araçları topluluğu. IP tarama, port kontrolü, DNS sorgulama gibi temel ağ işleri için geliştirildi.

---

## Özellikler

- IP adresi ve port tarama  
- DNS sorgulama  
- Ağ paket dinleme  
- Wordlist tabanlı parola denemeleri  
- Komut satırından kolay kullanım

---
